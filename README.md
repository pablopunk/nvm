# Nevermind (MVM)

An AI-native desktop command palette inspired by Raycast/Spotlight.

## MVP

- Electron + Vite + React + TypeScript
- Warm hidden palette window for instant display
- Global shortcut: `Option+Space` / `Alt+Space`
- App launcher indexing for macOS, Windows, and Linux
- URL opener, web-search fallback, and in-memory clipboard history
